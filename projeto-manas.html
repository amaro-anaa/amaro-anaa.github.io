Sistema MANA’s Lanche 🍔

Contexto

Projeto acadêmico desenvolvido com o objetivo de simular a digitalização de um pequeno negócio do setor alimentício.

Problema

O controle de pedidos era feito de forma manual, gerando desorganização, atrasos no atendimento e maior probabilidade de erros operacionais.

Solução

Foi desenvolvido um sistema utilizando low-code para centralizar os pedidos, organizar as informações e melhorar o fluxo de atendimento, com foco na estruturação de dados e facilidade de uso.

Resultados

  • Melhor organização dos pedidos
  • Redução de erros operacionais
  • Maior agilidade no atendimento

Funcionalidades

  • Cadastro de produtos
  • Registro de pedidos
  • Interface intuitiva
