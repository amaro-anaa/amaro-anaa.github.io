Sistema MANA’s Lanche 🍔

Sobre o projeto

Sistema desenvolvido para otimizar o gerenciamento de pedidos de uma lanchonete, tornando o processo mais rápido, organizado e digital.

Objetivo

Desenvolver um sistema (LowCode) de gerenciamento de pedidos para a lanchonete MANA’s Lanche, que permita registrar pedidos de forma eficiente, armazenar informações de pedidos e produtos.

Metodologia

O desenvolvimento será dividido em etapas semanais, seguindo o cronograma acima. Serão utilizadas ferramentas definidas pela equipe para controle de tarefas, documentação e versionamento do código. Cada fase terá entregas parciais para garantir o progresso contínuo e o alinhamento com os requisitos do cliente.

Funcionalidades

  • Cadastro de produtos
  • Registro de pedidos
  • Interface intuitiva
