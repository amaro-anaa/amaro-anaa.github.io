🍞 Padaria Pão Fresquim

Sobre o projeto

Desenvolvimento de uma solução para gerenciamento de uma padaria, com foco na organização de processos e controle de informações. O projeto envolveu análise de requisitos, modelagem do sistema e definição de funcionalidades voltadas para otimizar o atendimento e a gestão do negócio.

Funcionalidades

  • Cadastro de produtos
  • Controle de pedidos
  • Organização de informações
  • Apoio ao atendimento

Organização do projeto

O desenvolvimento do sistema foi estruturado em diferentes áreas, seguindo boas práticas de engenharia de software e metodologia ágil.

  • Gestão do projeto com base em Scrum
  • Levantamento e análise de requisitos
  • Modelagem e estruturação de dados
  • Desenvolvimento backend
  • Desenvolvimento frontend com foco em experiência do usuário
  • Integração fullstack
  • Testes e validação de qualidade
